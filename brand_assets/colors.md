# Brand Colors & Design System

## Colors

| Token | Hex | Usage |
|---|---|---|
| Background | `#070707` | Page background |
| Card | `#111111` | Card surfaces, elevated elements |
| Border | `#242424` | All borders at rest |
| Divider | `#1a1a1a` | Section dividers, subtle lines |
| Text primary | `#f0f0f0` | Headings, main content |
| Text secondary | `#c8c8c8` | Subheadings, descriptions |
| Text muted | `#424242` | Labels, metadata, inactive |
| Accent white | `#d8d8d8` | CTAs, primary interactive elements |
| **Accent amber** | `#b89060` | **ONLY: FC-2026, LASC 2026, Ignitia tags** |
| Amber bg | `#120e06` | Background behind amber elements |
| Amber border | `#28200e` | Border around amber elements |

## Typography

| Role | Font | Usage |
|---|---|---|
| Body | Inter | All body copy, paragraphs, descriptions |
| Technical | JetBrains Mono | Badges, labels, GPIO tables, code, technical specs |

## Rules

1. **No grid, no texture, no noise** — flat #070707 background only
2. **No glow, no blur, no gradients** — hard edges only
3. **Amber rule** — #b89060 appears ONLY on competition/project tags: FC-2026, LASC 2026, Ignitia-related. Never elsewhere.
4. **Monospace rule** — JetBrains Mono only on technical content. Never on body copy.
5. **Borders** — 1px solid #242424 at rest. Never soft or glowing.
6. **Hover state** — border goes to #2a2a2a. No color change.
7. **Placeholder images** — use `https://placehold.co/WIDTHxHEIGHT/111111/424242`
